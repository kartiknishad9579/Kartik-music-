### ✨ Features
- `/play <song name>` - YouTube se direct VC me stream
- `/pause` - Gaana pause karo
- `/resume` - Wapas chalu karo  
- `/stop` - VC se leave karo
- `/skip` - Agla gaana bajao
- High Quality Audio - 48000Hz
- Playlist & Queue Support
